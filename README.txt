1. Install SceneBuilder to be able to view the designs
	If not install from this link: https://gluonhq.com/products/scene-builder/

2. Start Scenebuilder
	- Click on File > Open ...
	- Navigate to the directory containing the unzipped files
	- Then select the fxml file you want to view from the scenebuilder_ui_files folder
	- This will open the design in Scenebuilder
